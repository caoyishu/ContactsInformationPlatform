# 前言

当你开始认识到这个问题的时候，你身边也快没有朋友了。

但凡你开始意识到这个问题的时候，你似乎可以开始尝试一下，使用一下本平台，记录一下你从小到大同学录，或者备忘录里的点点滴滴。
（如果你不介意在自己的机器安装一个MySQL数据库的话）。

也许美好的重逢，从一通电话，一条短信，或者一封诈骗邮件开始。
